Search public MSF-related publications, research metadata and humanitarian updates from open sources. This site retrieves records; it does not analyse, forecast or represent MSF.

Sources

Each source has a different public record scope. This site keeps retrieval separate from interpretation.

Live query

OpenAlex scholarly works

OpenAlex is an open catalogue of scholarly works. Use it to find MSF-related journal articles, papers, topics, authors and publication metadata.

What it can answer
  • What scholarly works mention MSF or a topic?
  • Which authors and journals appear in MSF-related research?
  • What recent publications are indexed?

Bibliographic metadata may not prove official MSF authorship or endorsement. Check the original publication.

Live query

Crossref scholarly metadata

Crossref provides public bibliographic metadata for scholarly publications. Use it as a second route for journal/article metadata and DOI lookup.

What it can answer
  • What publications match a topic or phrase?
  • Which DOI and journal metadata are available?

Crossref is a metadata index. It may not identify MSF affiliation cleanly for every record.

Optional live query - appname may be required

ReliefWeb reports and updates

ReliefWeb indexes humanitarian reports, updates, maps and operational information from humanitarian organisations, including MSF where items are syndicated there.

What it can answer
  • What MSF-related reports or updates appear on ReliefWeb?
  • What recent humanitarian updates match a country, crisis or topic?

Requires an approved appname in some cases. It contains syndicated humanitarian information, not all MSF publications.

About
